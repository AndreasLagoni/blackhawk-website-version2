    Let us introduce you to our achievement system

    This is something we are really proud of. The system is automated and something that makes us stand out. Let us introduce you to the system. Every person on the server will get these achievements, they are bound to each steam ID, so everyone will be able to climb the achievement ladder. Some achievements will contain titles, which you can use ingame so show it off! ALL of the achievements will give you points. There are a few achievements that are hidden and will only be available after you achieve it.

    Tier 1 is obviously the easiest
    From tier 1 to tier 2 there is big jump!
    From tier 2 to tier 3 there is an even bigger jump!
    Unique achievements are achievements that we thought needed their own tier. Some of these will contain titles and some wont. Happy hunting!
